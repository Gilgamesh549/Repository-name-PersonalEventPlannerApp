# Personal Event Planner App

## Overview
This Android application allows users to manage personal events such as work, social, and travel activities.

## Features
- Create, read, update, and delete events
- Local data persistence using Room Database
- Modern navigation with Bottom Navigation Bar
- Input validation and error handling

## Technologies Used
- Kotlin
- Room Database
- MVVM Architecture
- Jetpack Navigation

## How to Run
1. Open project in Android Studio
2. Build and run on emulator or device
